### 📝 À propos

Ce projet est une interprétation personnelle d'un index de liens sociaux (Social Link Index). Conçu pour être entièrement statique, il est rapide, sécurisé et facile à héberger sur des services comme GitHub Pages, Netlify ou Vercel.

---
### ✨ Fonctionnalités

Le template se distingue par plusieurs touches d'interactivité :

- **Effet Typewriter :** Un affichage dynamique du texte propulsé par JavaScript pour une introduction animée.

- **Arrière-plan Aléatoire :** Chaque visite offre une expérience visuelle unique grâce à un système de background randomisé.

- **Easter Eggs :** Des fonctionnalités cachées ont été glissées dans le code pour récompenser la curiosité des utilisateurs.

- **100% Statique :** Aucun backend requis, une simple gestion HTML/CSS/JS.

### 🚀 Installation & Personnalisation

Vous pouvez librement éditer ou forker ce dépôt pour l'adapter à vos besoins.

    Cloner le projet
    Bash

    git clone https://github.com/votre-utilisateur/votre-repo.git

    Configuration des liens
    Modifiez le fichier index.html pour y intégrer vos propres réseaux sociaux.

    Déploiement
    Glissez-déposez simplement les fichiers sur votre hébergeur statique favori.

### 🛠️ Technologies

    HTML5 / CSS3 (Structure et Design)

    JavaScript (Logique du Typewriter et Randomizer)
